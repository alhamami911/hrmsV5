Admas HRMS for Odoo 18

Admas HRMS is a fully integrated Human Resource Management System developed by AdmasTech for Odoo 18. Designed to serve modern businesses, it provides all the essential features required to manage human capital efficiently, with full support for the GCC region and Saudi Arabia's HR regulations.

Key Features

  • Employee records & contract management
  • Leave and attendance tracking
  • Document expiry notifications
  • Salary advance & loan requests
  • Fully integrated payroll system
  • Arabic & English user interface
  • Smart dashboards & analytical reports
  • Multi-company environment supported
